# PyDOS
Фан система написаная на питоне (не является реальной операционной системой).

## Команды
help - Пиши если забыл команды.
ver - Показывает версию.
time - Показывает дату и время с точностью до секунды.
cls - Просто очищает командную строку.
exit - Закрыть программу.
